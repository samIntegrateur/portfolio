En quelques mots

J’ai commencé à travailler dans le “digital” en 2010 en tant qu’intégrateur HTML. Depuis, beaucoup de choses ont évoluées : CSS3, HTML5, Responsive Web-design, pré-processeurs CSS, flex module, grid module, etc. Les nouveautés viennent régulièrement bousculer nos acquis, ce qui fait tout l’intérêt de ce secteur.

Pour mon plus grand bonheur, le terme “intégrateur” a des contours assez mal définis : parfois un peu UI Designer, d’autres plutôt développeur “front”, j'aime m'adapter au contexte dans lequel je travaille.

J’évolue progressivement vers un profil de Développeur Front-End : le Javascript moderne ainsi que les frameworks Angular, React suscitent principalement mon intérêt pour avoir un profil plus complet.
